For the latest and greatest version of this readme file you can visit the SVN repository:
SVN repository https://bagaturchess.svn.sourceforge.net/svnroot/bagaturchess


As a chess programmer,
you want to have the sources of Bagatur tools as a part of the distribution so that you can easily compile and run them inside the Eclipse development environment.
This source distribution is exactly what you want. It is archived eclipse workspace with simple main methods inside its sub-projects.
For more information you can have a look at the corresponding readme files of each sub-project.


Have a nice usage ... and feel free to contribute http://sourceforge.net/projects/bagaturchess/develop
